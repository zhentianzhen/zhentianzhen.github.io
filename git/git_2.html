git命令别名设置

      git命令敲多了，就觉得能少敲几个字还是多好的，那么现在就来看看怎样精简命令：

设置alias：

vim ~/.gitconfig

//写入以下内容

[alias]  
  st = status  
  ci = commit  
  br = branch  
  co = checkout  
  df = diff
  l = log

//顺个遍把git的颜色设置一下：

[color]
  status = auto
  diff = auto
  branch = auto
  interactive = auto

//保存退出，搞定
